LinkCoin 使用教程

    PC客户端加速

    教程适合windows上需要指定进程加速的用户，比如加速Telegram，Discord等等。也有个别程序不支持，比如微软的Edge浏览器。

    使用本客户端之前确保已经关闭ss或者ssr的代理功能，将系统代理模式改为直连，以确保不受其影响，如下图。

    下载程序

    • LinkCoin定制客户端下载
    • .net3.5环境下载

    如果打不开本程序请确认自身系统装有Microsoft .NET Framework 3.5，一般win7以上都是自带的无需额外下载安装。

    请确保截图地址栏圈出来的部分没有出现中文， 且文件夹内文件保持完整，不能只解压 EXE 文件。

    运行程序

    • 双击程序文件夹下的 LinkCoin.exe 运行程序，由于程序需要运行 Proxyfier 全局加速或者加速进程所以需要管理员权限，如弹出提示框请点击确认。

    • 打开程序后， 请使用网站注册邮箱和密码登录 (密码有符号的可能会登录失败请修改密码后登录)， 可以勾选记住密码， 下次打开不用再次输入， 登录成功需要网站内有 激活且流量未使用完的服务

    SS使用

    首次运行程序需要点击 环境安装 按钮，按钮变成灰色说明已完成相关程序的绿化安装。

    环境安装失败的解决办法:

    1.手动复制程序目录下的 system32 和 sysWOW64 文件夹覆盖到你的 C:\Windows.
    2.检查 %USERPROFILE%\AppData\Roaming\Proxifier\Profiles 这个目录下linkcoin.ppx 是否存在，没有的话就复制程序目录下的 riftxx.ppx 到这个目录下。
    3.分别运行程序目录 Proxifier下面的 SysSettings32.exe 和 SysSettings64.exe ，看下 install 是否是灰色状态，如果不是请点击 install 安装驱动。
    4.双击程序目录下的 linkcoin.reg 导入注册表，然后重新打开客户端即可。

    • 选择你需要使用的套餐， 在右侧选择你想要的连接方式(全局， 域名， 进程)。双击你想要使用的地区线路 即可使用 (任何规则的修改需要重新连接才能生效)。

    故障排除

    • 如果连接节点以后无法打开网站，请找到电脑右下角Proxyfier图标，双击图标查看弹出信息框底部是否有红色报错信息。

    • 如果出现报错信息，请双击proxifier文件夹下SysSettings64.exe(32位系统请使用SysSettings32.exe)，依次点击 uninstall,reset winsock settings,关闭后重新打开点击 install。最后打开客户端连接节点，查看是否正常。
